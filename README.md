# Secret Calculator

A modern and responsive calculator built using HTML, CSS, and JavaScript. Besides performing standard mathematical calculations, the application includes a hidden secret passcode feature that unlocks a predefined URL.

---

## Features

- Basic arithmetic operations
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Percentage

- Advanced operations
  - Square Root
  - Square
  - Reciprocal (1/x)

- Responsive user interface

- Keyboard support

- Input validation and error handling

- Hidden secret passcode functionality

- Modern glassmorphism-inspired UI

---

## Technologies Used

## How to Run

1. Download or clone the repository.

```
git clone <repository-url>
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation is required.
